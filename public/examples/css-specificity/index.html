⚖️ CSS 权重优先级综合演示

多个样式冲突时，谁赢了？——权重高的覆盖权重低的。

1️⃣ 权重 PK 擂台

同一个元素，多个选择器同时设置 color：

🏆 我是什么颜色？

div (蓝色) vs .box (绿色) vs #header (紫色) vs style="" (橙色) 

/* 多个选择器都设置了 color，谁赢？ */
div         { color: blue; }     /* 权重 0,0,0,1 */
.box        { color: green; }    /* 权重 0,0,1,0 — 比上面高 */
#header     { color: purple; }   /* 权重 0,1,0,0 — 比上面高 */
style=""                         /* 权重 1,0,0,0 — ✅ 最高，胜出！ */

2️⃣ 复合选择器权重累加

选择器权重计算过程
*0,0,0,0通配符
div0,0,0,11 个标签
.box0,0,1,01 个类
#header0,1,0,01 个 ID
.nav ul li a0,0,1,31 个类 + 3 个标签
#header .box0,1,1,01 个 ID + 1 个类
div#header0,1,0,11 个 ID + 1 个标签
style=""1,0,0,0行内样式
color: red !important∞无敌

3️⃣ 继承的权重为 0

继承来的样式，任何直接选中的样式都可以覆盖它：

我是 span，直接选中 → 红色

我是 p，直接选中 → 绿色

我是继承 div 的蓝色

<div style="color: blue;">
  <span style="color: red;">红色（直接选中）</span>
  <p style="color: green;">绿色（直接选中）</p>
  <p>蓝色（继承自 div）</p>
</div>

4️⃣ 同权重：后写的覆盖先写的

我是什么颜色？

都是类选择器，权重相同 → 后面的样式生效

/* 两个选择器权重完全一样 */
.same-weight { color: blue; }       /* 先写 */
.same-weight { color: #e74c3c; }    /* 后写 → 生效 */

📋 权重速查表

等级选择器权重
最高!important∞ 无敌
1 级行内样式 style=""1,0,0,0
2 级ID 选择器 #id0,1,0,0
3 级类 / 属性 / 伪类0,0,1,0
4 级标签 / 伪元素0,0,0,1
最低通配符 *0,0,0,0
—继承样式无权重（0）
💡 实用原则：
1. 权重冲突时 → 提高选择器权重，别用 !important
2. 同等权重 → 后写的覆盖先写的
3. 继承的权重是 0 → 任何直接选中都大于继承
4. 能用 class 解决的尽量用 class

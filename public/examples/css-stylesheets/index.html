🎨 三种样式表综合演示

同一个页面展示行内样式、内部样式、外部样式三种写法。

1️⃣ 行内样式 style 属性

样式直接写在标签的 style 属性中，只作用于这一个标签：

🔵 我喜欢蓝色（这是行内样式） 
<div style="color: #2980b9; font-size: 1.3em; font-weight: bold;
     background: #eaf2f8; padding: 0.8rem 1.2rem;
     border-radius: 8px; border-left: 4px solid #2980b9;">
  🔵 我喜欢蓝色
</div>

2️⃣ 内部样式 style 标签

样式写在 <style> 标签中，放在 <head> 里，作用于整个页面：

🔴 我喜欢红色（内部样式）

所有 <strong> 标签都会变成红色，因为内部样式写了 strong { color: #e74c3c; }

<style>
  strong {
    color: #e74c3c;
    font-size: 1.2em;
  }
</style>

3️⃣ 外部样式 .css 文件

样式写在独立的 .css 文件中，用 <link> 引入：

🟢 我喜欢绿色（外部样式）

所有 <p> 标签都会变成绿色，因为 style.css 里写了 p { color: #27ae60; }

<!-- HTML 中引入 -->
<link rel="stylesheet" href="./style.css">

/* style.css 文件内容 */
p {
  color: #27ae60;
  font-size: 1.1em;
}

4️⃣ 三种方式对比

对比项 行内样式 内部样式 外部样式
写法位置 标签 style 属性 <style> 标签 独立 .css 文件
复用范围 仅当前标签 当前页面 所有引入的页面
优先级 🔝 最高 🔼 中间 🔽 最低
推荐程度 ⭐ ⭐⭐ ⭐⭐⭐⭐⭐
💡 结论：实际项目开发中，优先使用外部样式——HTML 只管结构，CSS 只管样式，职责清晰，便于维护。

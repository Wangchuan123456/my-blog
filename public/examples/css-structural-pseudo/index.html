🔢 结构伪类选择器综合演示

根据元素在父元素中的位置来选中——:first-child、:last-child、:nth-child。

1️⃣ :first-child + :last-child

li:first-child — 第一个变红；li:last-child — 最后一个变蓝：

  • 第1个li ✅ first-child → 红色
  • 第2个li
  • 第3个li
  • 第4个li
  • 第5个li
  • 第6个li
  • 第7个li
  • 第8个li
  • 第9个li
  • 第10个li ✅ last-child → 蓝色
li:first-child { color: red; }
li:last-child  { color: blue; }

2️⃣ :nth-child(数字)

li:nth-child(3) 和 li:nth-child(8)：

  • 第1个li
  • 第2个li
  • 第3个li ✅ 选中 → 绿色
  • 第4个li
  • 第5个li
  • 第6个li
  • 第7个li
  • 第8个li ✅ 选中 → 紫色
  • 第9个li
  • 第10个li
li:nth-child(3) { color: green; }
li:nth-child(8) { color: purple; }

3️⃣ :nth-child(odd/even) 奇偶

:nth-child(odd) 奇数：

  • 第1个li（奇）
  • 第2个li
  • 第3个li（奇）
  • 第4个li
  • 第5个li（奇）
  • 第6个li
  • 第7个li（奇）
  • 第8个li
  • 第9个li（奇）
  • 第10个li

:nth-child(even) 偶数：

  • 第1个li
  • 第2个li（偶）
  • 第3个li
  • 第4个li（偶）
  • 第5个li
  • 第6个li（偶）
li:nth-child(odd)  { color: red; }   /* 奇数 */
li:nth-child(even) { color: blue; }  /* 偶数 */

4️⃣ :nth-child(an+b) 公式

n+3 — 第 3 个及以后：

  • 第1个li（前 2 个不变）
  • 第2个li（前 2 个不变）
  • 第3个li ✅ n+3 选中
  • 第4个li ✅ n+3 选中
  • 第5个li ✅ n+3 选中
  • 第6个li ✅ n+3 选中

-n+2 — 前 2 个：

  • 第1个li ✅ -n+2 选中
  • 第2个li ✅ -n+2 选中
  • 第3个li（后面的不变）
  • 第4个li
  • 第5个li
li:nth-child(n+3)  { color: red; }    /* 第3个及以后 */
li:nth-child(-n+2) { color: blue; }   /* 前 2 个 */
💡 公式理解：n 从 0 开始计数，n+3 → 3,4,5,6…；-n+2 → 2,1,0（0 无元素）。
2n = even（偶数），2n+1 = odd（奇数）。

5️⃣ 实战：表格隔行变色

表头蓝底白字 + 偶数行灰底 + 鼠标悬停高亮：

姓名得分日期
杨瀚森10分2025-07-12
易建联9分2007-10-31
王治郅6分2001-04-06
孙悦4分2008-12-08
周琦2分2018-10-31
姚明0分2002-10-31
/* 表头 */
.data tr:first-child { background-color: #8fbcf1; color: #fff; }

/* 偶数行变色 */
.data tr:nth-child(even) { background-color: #f9f9f9; }

/* 鼠标悬停 */
.data tr:hover { background-color: #f1f1f1; }

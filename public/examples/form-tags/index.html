📋 表单标签综合演示

展示各种表单控件的用法——文本框、密码框、单选框、复选框、文件上传、下拉菜单、文本域、按钮。

1️⃣ 综合注册表单

这是一个模拟注册表单，把所有控件放在一起展示：

📝 文本框 input text

  • （required 必填）

🔒 密码框 input password

  • （maxlength="10"）

🔘 单选框 input radio

  • （同一组 name 相同，checked 默认选中）

☑️ 复选框 input checkbox

📎 文件上传 input file

  • （multiple 多文件 + accept 限制类型）

📄 文本域 textarea

📋 下拉菜单 select + option

  • （selected 默认选中）

🎯 按钮 button / input submit

* 点击"重置"会清空所有输入恢复到初始状态

2️⃣ 各部分代码示例

文本框 + 密码框

<input type="text" placeholder="请输入账号" required autocomplete="off">
<input type="password" placeholder="请输入密码" maxlength="10" required>

单选框 + 复选框

<input type="radio" name="gender" value="male" checked> 男
<input type="radio" name="gender" value="female"> 女

<input type="checkbox" name="hobby" value="football"> 足球
<input type="checkbox" name="hobby" value="basketball"> 篮球

文件上传

<input type="file" multiple accept="image/*,video/*">

文本域

<textarea rows="4" cols="30"></textarea>

下拉菜单

<select name="city">
  <option value="beijing">北京</option>
  <option value="shanghai" selected>上海</option>
  <option value="guangzhou">广州</option>
</select>

按钮

<button type="submit">提交</button>
<button type="reset">重置</button>
<button type="button" disabled>禁用</button>
💡 提示：点击"重置"按钮会清空所有填写的内容，恢复成初始状态。 这是因为 type="reset" 的作用。

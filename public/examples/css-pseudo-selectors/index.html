🔗 分组选择器与链接伪类综合演示

1️⃣ 分组选择器 逗号 ,

多个选择器共用同一套样式：

这是 h3 标题（红色加粗）

这是 class="subtitle" 的段落（红色加粗）

这是 highlight-text 的 span（红色加粗）

普通段落没有变化——三个不同的选择器共用了同一样式。

/* 分组选择器：逗号隔开 */
.group-demo h3,
.group-demo .subtitle,
.group-demo .highlight-text {
  color: #e74c3c;
  font-weight: bold;
}

2️⃣ 链接伪类 LVHA

链接的四种状态，按 :link → :visited → :hover → :active 顺序写：

未访问的链接（蓝色）

已访问链接（紫色，点过 .md 页面后）

鼠标悬停试试（变红加下划线）

点击时（变绿）

/* LVHA 顺序 */
a:link    { color: #2980b9; text-decoration: none; }
a:visited { color: #8e44ad; }
a:hover   { color: #e74c3c; text-decoration: underline; }
a:active  { color: #27ae60; }
💡 记忆 LVHA：Love ❤️ HA — :link → :visited → :hover → :active

3️⃣ 开发推荐：简写方式

实际项目中最常用的写法，只写默认 + hover：

默认蓝色无下划线

鼠标悬停变红加下划线

/* 简写：a 本身设默认，a:hover 改交互 */
a {
  color: #2980b9;
  text-decoration: none;
}
a:hover {
  color: #e74c3c;
  text-decoration: underline;
}

4️⃣ :hover 也用于其他元素

鼠标移过来

← 鼠标移入变紫色，移出恢复

.hover-box {
  background: #e8f4fd;
  transition: 0.3s;
}
.hover-box:hover {
  background: #6c63ff;
  color: #fff;
}

📋 伪类汇总

伪类触发时机常用元素
:link未访问链接<a>
:visited已访问链接<a>
:hover鼠标悬停所有元素 ✅
:active正在点击<a>、按钮
:focus获得焦点<input> 等表单元素

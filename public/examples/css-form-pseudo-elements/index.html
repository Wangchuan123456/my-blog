🎯 表单伪类与伪元素综合演示

1️⃣ :disabled 禁用

← 禁用按钮透明度 0.4，鼠标变禁止符号

button:disabled {
  opacity: .4;
  cursor: not-allowed;
}

2️⃣ :checked 选中

input:checked + label {
  color: #ff6900;
  font-weight: bold;
}

3️⃣ ::first-line + ::first-letter

伪元素选择器：首行文字会变成橙色。首字母会放大变红。这段文字是用来展示效果的示例内容，请多写一些文字来看效果。

p::first-line  { color: #ff6900; }
p::first-letter { color: red; font-size: 28px; }

4️⃣ ::placeholder 占位文本

← 占位文字变红色小字

textarea::placeholder {
  color: red;
  font-size: 12px;
}

5️⃣ ::before + ::after

pink ← before 插入"我是 "，after 插入" 老师"

今日推荐 ← before 加🔥，after 加紫色下划线

/* 插入文字 */
.box::before { content: '我是 '; color: red; }
.box::after  { content: ' 老师'; color: red; }

/* 装饰图标 + 下划线 */
.title::before { content: '🔥'; }
.title::after  { content: ''; display: block; width: 60px; height: 3px; background: #6c63ff; }
⚠️ 注意：::before 和 ::after 必须设置 content 属性（可以是空字符串 ''），否则伪元素不会显示。 它们默认是行内元素，如果需设置宽高要加 display: block。

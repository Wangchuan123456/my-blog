5️⃣ 组合变换

组合效果
/* 组合：顺序影响结果 */
transform: translateX(100px) rotate(360deg) scale(1.2);

/* 搭配 transition 实现平滑动画 */
.box { transition: all .5s; }
.box:hover { transform: scale(1.1); }
💡 提示：transform 不会影响其他元素的位置，搭配 transition 是做悬停动效的黄金组合。

📍 CSS 定位布局综合演示

1️⃣ 相对定位 relative

相对定位
top:30 left:30
后面的盒子（不受影响）
position: relative;
top: 30px; left: 30px;  /* 相对自己原来位置偏移 */

2️⃣ 绝对定位 absolute

子绝父相：父 relative，子 absolute

子绝
.father { position: relative; }
.child { position: absolute; right: 20px; bottom: 30px; }

3️⃣ 固定定位 fixed

页面底部固定条（试试滚动页面）：

⬇ 固定底部 fixed（滚动也不动）
position: fixed;
left: 50%; margin-left: -150px;  /* 居中 */
bottom: 20px;

4️⃣ 粘性定位 sticky

粘性导航 — 滚动到顶部就粘住

向下滚动看看导航栏的效果……

当页面滚动到导航栏到达顶部时，它会粘住不动。

继续滚动……

导航栏依然固定在顶部。

position: sticky;
top: 0;   /* 到达顶部时固定 */

5️⃣ z-index 叠放次序

鼠标悬停图片，z-index 放大盖住其他：

img:hover { z-index: 1; }  /* 值越大越靠上 */
💡 核心口诀：子绝父相。z-index 只对定位元素生效。

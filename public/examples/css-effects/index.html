/* 谁做过渡给谁加 */
.card {
  transition: all 0.3s;           /* 所有属性都过渡 */
}
.card:hover {
  transform: translateY(-4px);    /* 向上抬升 */
  box-shadow: 0 8px 25px rgba(108,99,255,0.4);
}

3️⃣ 综合案例：AI 渐变按钮

.ai-button {
  box-shadow: 0 4px 15px rgba(0,0,0,0.1);
  transition: all 0.3s ease;
}
.ai-button:hover {
  transform: translateY(-2px);          /* 上移 */
  box-shadow: 0 6px 20px rgba(0,0,0,0.15);  /* 阴影加深 */
  background: linear-gradient(135deg, #3b5bdb, #1a365d); /* 颜色变化 */
}

4️⃣ 页面初始化 CSS Reset

这个页面顶部就用了初始化，看效果：

本页顶部已生效的初始化：

  • ✅ * { margin: 0; padding: 0; box-sizing: border-box; }
  • ✅ ul, ol { list-style: none; } — 看下面的列表没有圆点
  • ✅ a { text-decoration: none; }
  • ✅ input, button { outline: none; }

对比：

  • 首页 — 没有列表圆点
  • 登录 — 因为 list-style: none
  • 关于 — 干净清爽
/* 推荐放在 CSS 文件最前面 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

ul, ol { list-style: none; }
a { text-decoration: none; }
input, button { outline: none; }
img { display: block; }    /* 图片避免底部空隙 */
💡 实用组合拳：
盒子阴影 + 过渡 + 悬浮抬升 = 现代 UI 最常见的卡片交互效果。
页面初始化放在 CSS 文件最前面，一行代码解决浏览器样式差异。

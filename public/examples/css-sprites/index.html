🖼️ CSS 精灵图综合演示

1️⃣ 什么是精灵图？

多个小图标合成一张大图，每个元素只显示自己需要的那个小区域：

精灵图（多个图标在一张图里）

→

只显示红色

只显示绿色

只显示紫色

每个元素只显示需要的部分

2️⃣ 精灵图用法演示

通过 background-position 偏移，显示精灵图中不同的图标：

位置 (0, 0)
红色图标
位置 (-90px, 0)
绿色图标
位置 (-180px, 0)
紫色图标
位置 (-60px, -30px)
黄色图标
/* 基础设置：宽高 = 图标大小，精灵图作为背景 */
.sprite {
  width: 30px;                    /* 图标宽度 */
  height: 30px;                   /* 图标高度 */
  background: url(sprites.png) no-repeat;
}

/* 通过 background-position 显示不同图标 */
.sprite-1 { background-position: 0 0; }        /* 第1个 */
.sprite-2 { background-position: -90px 0; }    /* 第4个（左移90px） */
.sprite-3 { background-position: -180px 0; }   /* 第7个 */
.sprite-4 { background-position: -60px -30px; }/* 第2行第3个 */

3️⃣ 核心要点

要点说明
容器宽高= 要显示的图标宽高
background-image设置精灵图 + no-repeat
background-position**负值** 向左/上移动，显示需要的图标
x, y 坐标从设计工具中测量，或从精灵图提供方获取
💡 提示：现在更推荐用 字体图标（iconfont） 或 SVG 图标代替精灵图——矢量缩放清晰，颜色和大小随时可改。
但精灵图"合并请求"的思想仍然是性能优化的重要思路。

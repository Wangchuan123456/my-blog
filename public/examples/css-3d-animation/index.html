🎭 3D 变换与 animation 动画

1️⃣ 3D 旋转

rotateX
rotateY
transform: rotateX(180deg);  /* X轴翻转 */
transform: rotateY(180deg);  /* Y轴翻转 */

2️⃣ animation 动画

来回移动
旋转
弹跳
流光边框
@keyframes move {
  0%   { transform: translateX(0); }
  100% { transform: translateX(200px); border-radius: 50%; }
}
.box { animation: move 2s linear infinite alternate; }

3️⃣ animation 完整写法

.box {
  animation: move 2s ease 0s infinite alternate;
  /*          ↑名称 ↑时长 ↑曲线 ↑延迟 ↑次数 ↑方向 */
}
💡 区别：transition 需要鼠标触发，animation 自动播放。复杂动画用 @keyframes + animation。

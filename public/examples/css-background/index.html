🎨 CSS 盒子背景综合演示

1️⃣ background-color

pink
#6c63ff
rgba 半透明
background-color: pink;
background-color: #6c63ff;
background-color: rgba(108, 99, 255, 0.3);

2️⃣ background-image + background-repeat

repeat 默认
no-repeat
repeat-x
repeat-y
background-repeat: repeat;     /* 默认，全平铺 */
background-repeat: no-repeat;  /* 只显示一张 */
background-repeat: repeat-x;   /* 水平平铺 */
background-repeat: repeat-y;   /* 垂直平铺 */

3️⃣ background-position 位置

left top
center center
right bottom
/* 方位名词 */
background-position: left top;        /* 左上 */
background-position: center center;   /* 居中 */
background-position: right bottom;    /* 右下 */

/* 像素值 */
background-position: 50px 10px;

/* 百分比 */
background-position: 50% 50%;

4️⃣ background-size 尺寸

cover 铺满
contain 完整
50px 固定
background-size: cover;      /* 铺满盒子（可能被裁剪） */
background-size: contain;    /* 完整显示（可能有留白） */
background-size: 200px;      /* 指定宽度 */

5️⃣ background 复合写法

复合写法
渐变背景
/* 复合写法：color image repeat position/size */
background: pink url(ldh.png) no-repeat center center/200px;

/* 渐变 */
background: linear-gradient(135deg, #6c63ff, #a29bfe);

6️⃣ 渐变 linear-gradient

默认方向
向右
45deg
多色渐变
background: linear-gradient(pink, red);               /* 从上到下 */
background: linear-gradient(to right, pink, red);      /* 向右 */
background: linear-gradient(45deg, pink, red);         /* 45度 */
background: linear-gradient(90deg, #0096FF, #BB64FF, #F2416B, #EB7500);  /* 多色 */

7️⃣ 渐变按钮实战

button {
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
  border: none; color: #fff; padding: 10px 24px;
  border-radius: 6px; cursor: pointer;
}
💡 复合写法顺序：
background: color image repeat position/size attachment;
其中 position/size 之间用 斜杠 / 分隔，这是最容易写错的地方！

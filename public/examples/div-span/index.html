📦 <div> 与 <span> 综合演示

看 <div> 和 <span> 在实际页面中怎么用，和语义标签有什么区别。

1️⃣ <div> 布局容器

三个 <div> 并排展示（用 flex 布局排列），每个 <div> 都是一个独立的区块：

📌 区块 A

这是第一个 div

📌 区块 B

这是第二个 div

📌 区块 C

这是第三个 div

← 每个灰色虚线框都是一个 <div>，用来分组内容

HTML 结构：

<div style="display: flex;">          <!-- 外层 div：flex 容器 -->
  <div class="div-box">               <!-- 里层 div：区块 A -->
    <h4>📌 区块 A</h4>
    <p>这是第一个 div</p>
  </div>
  <div class="div-box">               <!-- 里层 div：区块 B -->
    <h4>📌 区块 B</h4>
    <p>这是第二个 div</p>
  </div>
  <div class="div-box">               <!-- 里层 div：区块 C -->
    <h4>📌 区块 C</h4>
    <p>这是第三个 div</p>
  </div>
</div>

2️⃣ <span> 修饰文本

<span> 可以包住部分文字，单独改变样式：

🔴 改变颜色： 今天天气 非常炎热，注意防暑。

💰 价格效果： 原价 ¥199 ¥99

⭐ 高亮标记： 这是一条 重要消息，请仔细阅读。

🏷️ 图标标记： ★ 推荐 ★ 热门 ● 新品

HTML 结构：

<p>
  原价 <span style="text-decoration: line-through; color: #999;">¥199</span>
  <span style="color: red; font-weight: bold; font-size: 1.4em;">¥99</span>
</p>

3️⃣ 🤝 <div> + <span> 组合实战

商品卡片：<div> 做整体布局，<span> 修饰价格文字：

🎧 无线蓝牙耳机

限时特惠
¥299 ¥129

续航 30 小时 · 降噪版 · 现货直发

HTML 结构：

<div class="card">                          <!-- div：整个卡片容器 -->
  <h3>🎧 无线蓝牙耳机</h3>
  <div class="price">
    <span class="tag">限时特惠</span>        <!-- span：标签 -->
  </div>
  <div class="price">
    <span class="old">¥299</span>           <!-- span：旧价（删除线） -->
    <span class="current">¥129</span>        <!-- span：现价（红色加粗） -->
  </div>
  <p class="desc">续航 30 小时 · 降噪版 · 现货直发</p>
</div>

4️⃣ 🔄 语义标签 vs 无语义标签

同样的页面结构，左边用语义标签，右边全部用 <div>——效果一样，但含义不同：

✅ 语义标签

页头
<header>
导航
<nav>
主内容
<main>
页脚
<footer>

实际渲染效果：

📰 网站标题
首页 关于 联系

页头
<div class="header">
导航

主内容
<div class="main">
页脚
<div class="footer">

💡 结论：两种写法在浏览器里看起来完全一样。区别在于—— 语义标签让代码"自解释"（看标签就知道是什么区域）， 无语义标签完全靠 class 名字来区分，可读性差一些。 所以有合适的语义标签时优先用语义标签。

🎨 字体样式与文本布局综合演示

文字相关的 CSS 属性——颜色、大小、粗细、对齐、缩进、行高等。

1️⃣ 字体颜色 color

💗 color: pink（颜色名称）

💚 color: #2CB5A5（十六进制，最常用）

❤️ color: rgb(255, 0, 0)（RGB）

💔 color: rgba(255, 0, 0, 0.5)（半透明）

/* 四种写法 */
color: pink;
color: #2CB5A5;         /* ✅ 推荐 */
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5);

2️⃣ 字体大小 font-size + 粗细 font-weight

font-size: 12px（小字）

font-size: 14px（稍大）

font-size: 16px（默认大小）

font-size: 20px（标题感）

font-weight: 400（正常粗细）

font-weight: 700（加粗）

font-weight: bold（也是加粗）

font-size: 16px;      /* 字号 */
font-weight: 400;     /* 正常粗细 */
font-weight: 700;     /* 加粗 */

3️⃣ 字体样式 font-style + 字体族 font-family

font-style: italic（斜体）

em 标签用 normal 取消斜体

font-family: "宋体"（宋体）

Microsoft YaHei（微软雅黑）

font-style: italic;    /* 斜体 */
font-family: "宋体";    /* 指定字体 */
font-family: "Microsoft YaHei", "PingFang SC", sans-serif;  /* 备选 */

4️⃣ 文本装饰 text-decoration

下划线 underline

上划线 overline

删除线 line-through

链接去掉下划线 none

text-decoration: underline;      /* 下划线 */
text-decoration: none;           /* 去掉装饰（a 标签常用） */
text-decoration: line-through;   /* 删除线 */

5️⃣ 文本对齐 text-align + 缩进 text-indent

左对齐（默认）
居中对齐
右对齐

text-indent: 2em — 这是首行缩进两个汉字的段落效果。 写中文文章时经常用到这个属性，让段落开头空两格，更符合阅读习惯。 

text-align: center;    /* 水平居中 */
text-indent: 2em;      /* 首行缩进两个汉字 */

6️⃣ 行高 line-height + 单行垂直居中

line-height: 1.5 — 这是正常行高。文字行与行之间的距离适中， 阅读起来比较舒服。这是推荐的写法。

line-height: 2.0 — 这是两倍行高。行间距更大， 适合需要留白较多的场景。

⬆ line-height = height = 50px → 单行文本垂直居中 
line-height: 1.5;          /* 相对值，推荐 */
line-height: 50px;         /* 等于容器高度 → 单行垂直居中 */

7️⃣ 字间距 letter-spacing + font 简写

letter-spacing: 5px（字间距加大）

letter-spacing: -1px（字间距收缩）

font 简写 — italic 700 18px/1.8 "Microsoft YaHei" 

letter-spacing: 2px;        /* 字间距 */
font: italic 700 18px/1.8 "微软雅黑";  /* 简写 */

📋 属性汇总

属性 作用 常用值
color文字颜色#2CB5A5、red
font-size文字大小16px、14px
font-weight文字粗细400、700
font-style斜体normal、italic
font-family字体族"宋体"
text-decoration装饰线none、underline
text-align水平对齐left、center
text-indent首行缩进2em
line-height行高1.5、26px
letter-spacing字间距2px
font简写italic 700 16px/1.5 "宋体"
💡 单行垂直居中技巧：设置 line-height 等于容器 height，文字就会自动垂直居中。 这是 CSS 中最常用的技巧之一。

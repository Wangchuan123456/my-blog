🎯 CSS 选择器综合演示

四种基础选择器的用法对比——类型、类、ID、通配符。

1️⃣ 类型选择器 p / strong

选中所有同名标签：

✅ 我是 <p> 标签，所有 p 标签都变成了绿色。

✅ 我也是 <p> 标签，同样变成绿色。

🔴 我是 <strong> 标签，所有 strong 都变成了红色。
🔴 我也是 <strong>，同样是红色。 
/* CSS：类型选择器 */
p { color: #27ae60; }
strong { color: #e74c3c; }

2️⃣ 类选择器 .class

选中所有带此 class 的元素，最灵活：

这是普通段落，没有样式。

🎯 我有 class="highlight"，背景高亮！

📦 卡片 1 — class="card-item"
📦 卡片 2 — class="card-item dark"（两个 class）
/* CSS：类选择器 */
.highlight { background: #fff3cd; font-weight: bold; }
.card-item { background: #eaf2f8; border: 1px solid #2980b9; }
.card-item.dark { background: #d5e8f5; }  /* 同时有两个 class 的元素 */
<!-- 一个元素可以有多个 class -->
<div class="card-item dark">卡片 2</div>

3️⃣ ID 选择器 #id

选中唯一的元素，一个页面中 id 不能重复：

⭐ 我是 id="special-box"，页面中唯一，紫色渐变背景！
/* CSS：ID 选择器 */
#special-box {
  background: linear-gradient(135deg, #6c63ff, #a29bfe);
  color: #fff;
  font-weight: bold;
  padding: 1rem 1.5rem;
  border-radius: 8px;
}

4️⃣ 通配符选择器 *

选中所有元素，常用于全局重置：

这个页面的所有元素都应用了 * { box-sizing: border-box; }

/* CSS：通配符选择器 */
* {
  box-sizing: border-box;  /* 所有元素统一盒子模型 */
}

5️⃣ 权重对比演示

当多个选择器冲突时，权重高的生效：

我是 <p>，类型选择器让我变绿。

我有 class="blue-text"，类选择器权重更高，所以我变蓝。

/* 类型选择器权重 0,0,1 */
p { color: #27ae60; }

/* 类选择器权重 0,1,0 — 更高，所以覆盖上面的绿色 */
.blue-text { color: #2980b9; }

6️⃣ 四种选择器总对比

选择器 写法 选中范围 权重 适用场景
通配符 * 所有元素 0,0,0 全局重置
类型 p 所有同名标签 0,0,1 全局默认样式
类 .class 所有带此 class 的元素 0,1,0 日常最常用
ID #id 唯一元素 1,0,0 页面唯一区块
💡 建议：日常开发中 80% 用类选择器就够了——复用性强、优先级适中、写起来也方便。 ID 选择器和通配符在特定场景再用。

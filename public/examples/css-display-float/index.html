1️⃣ display 显示类型

block — 块级（独占一行，可设宽高）：

block 1
block 2（每个都独占一行）

inline — 行内（不换行，不可设宽高）：

行内 1 行内 2 行内 3（并排显示）

inline-block — 不换行，但可设宽高：

A B C

三个 inline-block 并排，每个都设了 100×60px

display: block;          /* 块级：独占一行，可设宽高 */
display: inline;         /* 行内：不换行，宽高无效 */
display: inline-block;   /* 行内块：不换行，可设宽高 */
display: none;           /* 隐藏：不占位 */

  clear: both;
}

/* 或者父元素加 overflow */
.father { overflow: hidden; }

4️⃣ 图文环绕

图片

这是文字环绕效果。图片左浮动后，文字会自动环绕在图片的右侧和下方。这是浮动最初的设计目的——实现图文环绕排版。在没有浮动之前，要让文字环绕图片非常困难。浮动使得文字能够自然地在图片周围排列。

img {
  float: left;           /* 图片左浮动 */
  margin-right: 10px;    /* 与文字保持间距 */
}

5️⃣ display: inline-block vs float 区别

对比inline-blockfloat
是否脱离文档流❌ 不脱离✅ 脱离

并排效果✅ 并排✅ 并排
图文环绕❌ 不支持✅ 原生支持

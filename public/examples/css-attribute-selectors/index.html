🔍 属性选择器综合演示

五种属性选择器写法——有属性、等于、开头、结尾、包含。

1️⃣ [attr] 包含属性

a[class] — 有 class 属性的 a 标签变红：

没有 class（不变） 有 class="font" ✅ 变红 有 class="red" ✅ 变红 
a[class] { color: red; }

2️⃣ [attr="value"] 完全匹配

a[class="font"] — class 正好等于 font 才变红：

class="font" ✅ 完全匹配，变红 class="font14" ❌ 不等，不变 class="red" ❌ 不变 
a[class="font"] { color: red; }

3️⃣ [attr^="value"] 以 xx 开头

a[class^="font"] — class 以 font 开头的变红：

font ✅ 以 font 开头，变红 font14 ✅ 以 font 开头，变红 font16 ✅ 以 font 开头，变红 red ❌ 不以 font 开头，不变 
a[class^="font"] { color: red; }

4️⃣ [attr$="value"] 以 xx 结尾

a[class$="14"] — class 以 14 结尾的变红：

font14 ✅ 以 14 结尾，变红 font16 ❌ 以 16 结尾，不变 red14 ✅ 以 14 结尾，变红 
a[class$="14"] { color: red; }

5️⃣ [attr*="value"] 包含 xx

a[class*="font"] — class 中含有 font 的变红：

font ✅ 包含 font font14 ✅ 包含 font myfont ✅ 包含 font red ❌ 不包含 font 
a[class*="font"] { color: red; }

6️⃣ 实际应用：表单输入框

用 input[type="text"] 和 input[type="password"] 选中：

复选框（不会被选中）

/* 不用加 class，直接根据 type 属性选中 */
input[type="text"],
input[type="password"] {
  outline: none;
  border: 1px solid #ccc;
  border-radius: 4px;
}
input[type="text"]:focus,
input[type="password"]:focus {
  border-color: #6c63ff;
  box-shadow: 0 0 0 2px rgba(108,99,255,0.15);
}
💡 小结：属性选择器最大的好处是不用加 class——根据元素已有的属性就能选中，
特别适合表单控件、外部链接等场景。^ 开头、$ 结尾、* 包含，这三个最常用。

📰 多栏布局与鼠标样式

1️⃣ 多栏布局 column

CSS 多栏布局（Multi-column Layout）允许将内容分成多栏排列，就像报纸和杂志的排版方式一样。它通过 column-count 属性指定栏数，通过 column-gap 控制栏间距，通过 column-rule 添加栏之间的分割线。

这种布局方式特别适合大段文字内容，比如文章正文、用户协议、产品介绍等。将长文本分成多栏可以提高阅读效率，减少眼睛左右扫视的距离。

需要注意的是，如果子元素有固定高度或包含图片，可能需要使用 break-inside: avoid-column 来防止内容被分割到不同栏中。

.box { column-count: 3; column-gap: 20px; column-rule: 1px dashed red; }

2️⃣ 鼠标样式 cursor

样式CSS效果
默认default↖ 默认箭头
手型pointer👆 手型（可点击）
文本textI 型竖线
禁止not-allowed🚫 禁止点击

/* 最常用的两个 */
button { cursor: pointer; }
button:disabled { cursor: not-allowed; }
💡 最常用的 cursor：pointer（可点击）和 not-allowed（禁用）。工作中 80% 的场景就这两个。

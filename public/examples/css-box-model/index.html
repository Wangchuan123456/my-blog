📦 CSS 盒子模型综合演示

1️⃣ 边框 border

solid 实线
dashed 虚线
dotted 点线
border: 2px solid red;
border: 2px dashed red;
border: 2px dotted red;

单独设置各边：

四条边不同样式 
border-top: 3px solid #ff8400;
border-bottom: 1px dotted green;
border-left: 4px dashed blue;
border-right: 5px solid pink;

2️⃣ 圆角 border-radius

8px
50% 圆形
胶囊按钮
border-radius: 8px;     /* 小圆角 */
border-radius: 50%;    /* 圆形（正方形） */
border-radius: 18px;   /* 胶囊（height=36px一半） */

3️⃣ 内边距 padding

padding: 10px 20px（上下10px，左右20px） 
/* 简写 1~4 个值 */
padding: 10px;           /* 四个方向相同 */
padding: 10px 20px;      /* 上下 / 左右 */
padding: 10px 20px 30px; /* 上 / 左右 / 下 */
padding: 10px 20px 30px 40px;  /* 上 右 下 左（顺时针） */

/* 单独设置 */
padding-left: 10px;

4️⃣ 外边距 margin + 水平居中

margin: 0 auto → 水平居中

块级元素设固定宽度后，左右 auto 自动居中

.center {
  width: 300px;
  margin: 0 auto;   /* 水平居中 */
}

5️⃣ box-sizing

content-box
width=160
实际=160+15×2+5×2=200px
border-box ✅
width=160
实际=160px（内容被压缩） 
/* content-box（默认）— padding+border 撑大盒子 */
box-sizing: content-box;

/* border-box — padding+border 向内挤，width 就是实际宽度 */
box-sizing: border-box;   /* ✅ 推荐全局设置 */
💡 推荐全局设置：* { box-sizing: border-box; }
这样所有盒子的 width 就是实际可见宽度，不用再手动计算。

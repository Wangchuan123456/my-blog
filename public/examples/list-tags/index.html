📋 列表标签综合演示

展示无序列表 <ul>、有序列表 <ol>、描述列表 <dl> 的用法。

1️⃣ 无序列表 <ul>

顺序不重要的列表，默认前面显示圆点。

基本用法

  • 苹果
  • 香蕉
  • 橘子
  • 西瓜
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橘子</li>
  <li>西瓜</li>
</ul>

实际应用：导航菜单

用 <ul> 做导航菜单是非常常见的模式：

  • 首页
  • 文章
  • 项目
  • 关于
<nav>
  <ul>
    <li><a href="#">首页</a></li>
    <li><a href="#">文章</a></li>
    <li><a href="#">项目</a></li>
    <li><a href="#">关于</a></li>
  </ul>
</nav>

嵌套列表

<li> 里面可以再放一个 <ul>，形成多级菜单：

  • 水果
    • 苹果
    • 香蕉
  • 蔬菜
    • 白菜
    • 萝卜

2️⃣ 有序列表 <ol>

顺序重要的列表，默认显示数字编号。

基本用法

  1. 打开编辑器
  2. 编写 HTML 代码
  3. 保存为 .html 文件
  4. 用浏览器打开预览

type 属性：改变编号样式

通过 type 属性可以切换不同的编号格式：

type="1"（默认）

  1. 第一项
  2. 第二项
  3. 第三项

type="A"

  1. 第一项
  2. 第二项
  3. 第三项

type="a"

  1. 第一项
  2. 第二项
  3. 第三项

type="I"

  1. 第一项
  2. 第二项
  3. 第三项

type="i"

  1. 第一项
  2. 第二项
  3. 第三项

start 属性：自定义起始值

  1. 第五项（从 5 开始）
  2. 第六项
  3. 第七项

reversed 属性：倒序

  1. 第三名（排最前）
  2. 第二名
  3. 第一名（排最后）

3️⃣ 描述列表 <dl>

"术语 — 说明" 配对展示。

基本用法

HTML
超文本标记语言，负责网页的结构。
CSS
层叠样式表，负责网页的样式。
JavaScript
脚本语言，负责网页的交互。
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，负责网页的结构。</dd>

  <dt>CSS</dt>
  <dd>层叠样式表，负责网页的样式。</dd>

  <dt>JavaScript</dt>
  <dd>脚本语言，负责网页的交互。</dd>
</dl>

一个 dt 对应多个 dd

前端开发
网页的用户界面部分。
主要使用 HTML、CSS、JavaScript。
常用框架有 React、Vue、Angular。

4️⃣ 三种列表组合使用

实际开发中经常混合使用。下面是"学习路线"示例：有序列表做步骤，无序列表做子项，描述列表做说明：

  1. 第一阶段：HTML + CSS
    • HTML 基础标签
    • CSS 选择器与样式
    • Flex 布局
  2. 第二阶段：JavaScript
    • 变量与数据类型
    • DOM 操作
    • 事件处理
  3. 第三阶段：框架
    推荐框架
    React — 组件化开发
    Vue — 上手简单
💡 总结：
  • 顺序不重要 → <ul>
  • 顺序重要 → <ol>，配合 type / start / reversed
  • 术语 + 说明 → <dl>
  • 三种列表可以互相嵌套，灵活组合

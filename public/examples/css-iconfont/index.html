🔤 字体图标综合演示

使用 iconfont.cn 的图标——像文字一样控制大小和颜色。

1️⃣ 图标展示

引入 iconfont.css 后，用 class="iconfont icon-xxx" 使用：

icon-good
icon-xihuan
icon-arrow-right
icon-notification
icon-picture
icon-loading
<!-- 引入 CSS -->
<link rel="stylesheet" href="./iconfont/iconfont.css">

<!-- 使用图标 -->
<span class="iconfont icon-good"></span>

<!-- 修改样式 -->
<span class="iconfont icon-good" style="font-size: 40px; color: red;"></span>

2️⃣ 不同大小

用 font-size 控制图标大小：

.icon-good { font-size: 32px; color: #6c63ff; }

3️⃣ 不同颜色

用 color 控制图标颜色：

.iconfont { color: #e74c3c; }  /* 改颜色就像改文字颜色一样简单 */

4️⃣ 实际应用

导航菜单：

首页 消息 图片 收藏

按钮图标：

<nav>
  <span><i class="iconfont icon-home"></i> 首页</span>
  <span><i class="iconfont icon-search"></i> 搜索</span>
</nav>

<button><i class="iconfont icon-download"></i> 下载</button>

5️⃣ 引入方式说明

方式一：link 引入 CSS 文件（推荐）

<link rel="stylesheet" href="./iconfont/iconfont.css">

方式二：@import 在 CSS 中引入

@import url(./iconfont/iconfont.css);

目录结构：

项目目录/
├── index.html
├── iconfont/
│   ├── iconfont.css   ← 引入这个
│   ├── iconfont.ttf    ← 字体文件（css 内部引用）
│   ├── iconfont.woff
│   └── iconfont.svg
💡 注意事项：
1. iconfont 基础 class **必须加**，否则图标不显示
2. 图标类名以 iconfont.cn 上定义的为准（如 icon-good）
3. iconfont.css 和字体文件必须在同一目录
4. 修改图标大小用 font-size，颜色用 color

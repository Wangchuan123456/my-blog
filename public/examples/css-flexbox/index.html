📐 Flex 弹性盒布局综合演示

1️⃣ 基本 flex 布局

父元素加 display: flex，子元素自动水平排列：

1
2
3
4
← display: flex → 子元素自动并排
.container { display: flex; }

2️⃣ justify-content 主轴对齐

justify-content: flex-start（默认左对齐）

1
2
3

justify-content: center（居中）

1
2
3

justify-content: space-between（两端对齐）

1
2
3

justify-content: space-evenly（间距相等）

1
2
3
justify-content: flex-start;    /* 左对齐 */
justify-content: center;        /* 居中 */
justify-content: space-between; /* 两端对齐 */
justify-content: space-evenly;  /* 间距相等 */

3️⃣ align-items 交叉轴对齐

align-items: center（垂直居中） — 不同高度的项目：

1
2（高）
3（矮）

align-items: flex-end（底部对齐）

1
2（高）
3（矮）
align-items: center;      /* 垂直居中 */
align-items: flex-end;    /* 底部对齐 */
align-items: stretch;     /* 拉伸填满（默认） */

4️⃣ 完美居中

🎯 完美居中
display: flex + justify-content: center + align-items: center
/* 完美居中——Flex 最经典的应用 */
.container {
  display: flex;
  justify-content: center;   /* 水平居中 */
  align-items: center;       /* 垂直居中 */
}

5️⃣ flex: 1 等分

每个项目 flex: 1，等分容器宽度：

flex:1
flex:2
flex:1
中间项目 flex:2，宽度是两侧的两倍
.item { flex: 1; }       /* 等分剩余空间 */
.item2 { flex: 2; }     /* 其他项目的两倍 */

6️⃣ flex-direction 方向

flex-direction: column（垂直排列）

1
2
3
flex-direction: row;         /* 默认：水平 */
flex-direction: column;      /* 垂直排列 */
flex-direction: row-reverse; /* 水平反向 */

7️⃣ flex-wrap 换行

flex-wrap: wrap — 项目超出容器宽度时自动换行：

1 (120px)
2 (120px)
3 (120px)
4 (120px)
5 (120px)
flex-wrap: nowrap;   /* 默认：不换行 */
flex-wrap: wrap;     /* 换行 */
💡 一句话记住 Flex：
父元素 display: flex → 子元素水平排列
justify-content 控制水平方向 → align-items 控制垂直方向
flex: 1 等分空间 → 这就是 Flex 的核心用法 🎯

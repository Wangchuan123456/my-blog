## 前言

列表是网页中最常见的元素之一——导航菜单、文章目录、商品列表、术语解释……到处都是列表。

HTML 提供了三种列表标签，各自适用不同场景：

| 列表类型 | 标签 | 典型样式 | 用途 |
|---------|------|---------|------|
| **无序列表** | `<ul>` + `<li>` | ● 圆点 | 顺序不重要的列表 |
| **有序列表** | `<ol>` + `<li>` | 1. 2. 3. | 顺序重要的列表 |
| **描述列表** | `<dl>` + `<dt>` + `<dd>` | 术语 → 说明 | 键值对、术语解释 |

## 一、无序列表 &lt;ul&gt;

`<ul>` 表示 **Unordered List（无序列表）**，列表项之间**没有先后顺序**。

```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橘子</li>
</ul>
```

浏览器显示效果：

```
● 苹果
● 香蕉
● 橘子
```

每个列表项用 `<li>`（List Item）包裹。`<ul>` 的直接子元素只能是 `<li>`。

### 嵌套列表

列表里面可以再套列表——**`<li>` 里面可以再放一个 `<ul>`**：

```html
<ul>
  <li>水果
    <ul>
      <li>苹果</li>
      <li>香蕉</li>
    </ul>
  </li>
  <li>蔬菜
    <ul>
      <li>白菜</li>
      <li>萝卜</li>
    </ul>
  </li>
</ul>
```

显示效果：

```
● 水果
    ● 苹果
    ● 香蕉
● 蔬菜
    ● 白菜
    ● 萝卜
```

### 常见用途

- 导航菜单（`<nav>` 里的 `<ul>`）
- 文章分类列表
- 商品列表
- 任何顺序无关紧要的列表

## 二、有序列表 &lt;ol&gt;

`<ol>` 表示 **Ordered List（有序列表）**，列表项之间有明确的**先后顺序**。

```html
<ol>
  <li>第一步：打开编辑器</li>
  <li>第二步：编写 HTML</li>
  <li>第三步：保存并预览</li>
</ol>
```

浏览器显示效果：

```
1. 第一步：打开编辑器
2. 第二步：编写 HTML
3. 第三步：保存并预览
```

### type 属性：改变编号样式

`<ol>` 的 `type` 属性可以改变编号的样式：

| type 值 | 样式 | 示例 |
|---------|------|------|
| `1`（默认） | 阿拉伯数字 | 1. 2. 3. |
| `a` | 小写字母 | a. b. c. |
| `A` | 大写字母 | A. B. C. |
| `i` | 小写罗马数字 | i. ii. iii. |
| `I` | 大写罗马数字 | I. II. III. |

```html
<ol type="A">
  <li>第一项</li>
  <li>第二项</li>
  <li>第三项</li>
</ol>
```

显示效果：

```
A. 第一项
B. 第二项
C. 第三项
```

### start 属性：自定义起始值

```html
<ol start="5">
  <li>第五项</li>
  <li>第六项</li>
  <li>第七项</li>
</ol>
```

显示效果：

```
5. 第五项
6. 第六项
7. 第七项
```

### reversed 属性：倒序排列

```html
<ol reversed>
  <li>第三项（排最前）</li>
  <li>第二项</li>
  <li>第一项（排最后）</li>
</ol>
```

显示效果：

```
3. 第三项（排最前）
2. 第二项
1. 第一项（排最后）
```

### 常见用途

- 排行榜
- 操作步骤 / 教程
- 比赛名次
- 任何有顺序要求的内容

## 三、描述列表 &lt;dl&gt;

`<dl>` 表示 **Description List（描述列表）**，由"术语（term）— 描述（description）"对组成。

- `<dt>` — Description Term（术语/标题）
- `<dd>` — Description Details（描述/说明）

```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，负责网页的结构。</dd>

  <dt>CSS</dt>
  <dd>层叠样式表，负责网页的样式。</dd>

  <dt>JavaScript</dt>
  <dd>脚本语言，负责网页的交互。</dd>
</dl>
```

浏览器显示效果：

```
HTML
    超文本标记语言，负责网页的结构。
CSS
    层叠样式表，负责网页的样式。
JavaScript
    脚本语言，负责网页的交互。
```

### 一个 dt 对应多个 dd

一个术语可以有多个描述：

```html
<dl>
  <dt>前端</dt>
  <dd>网页的用户界面部分。</dd>
  <dd>包括 HTML、CSS、JavaScript。</dd>
</dl>
```

### 常见用途

- 术语表 / 词典
- 键值对数据（如商品规格）
- 元数据列表（如作者、发布时间）

## 四、三种列表的嵌套使用

实际开发中，三种列表可以组合使用。比如一个"操作步骤 + 说明"的混合结构：

```html
<ol>
  <li>
    <p><strong>安装 Node.js</strong></p>
    <ul>
      <li>前往官网下载</li>
      <li>按提示安装</li>
    </ul>
  </li>
  <li>
    <p><strong>创建项目</strong></p>
    <ul>
      <li>打开终端</li>
      <li>运行 <code>npm init</code></li>
    </ul>
  </li>
</ol>
```

## 小结

| 标签 | 含义 | 默认样式 | 适用场景 |
|------|------|---------|---------|
| `<ul>` | 无序列表 | ● 圆点 | 导航、分类、商品列表 |
| `<ol>` | 有序列表 | 1. 2. 3. | 步骤、排行、目录 |
| `<dl>` | 描述列表 | 缩进展示 | 术语表、键值对 |
| `<li>` | 列表项 | — | ul 和 ol 的子元素 |
| `<dt>` | 描述术语 | — | dl 的子元素 |
| `<dd>` | 描述详情 | — | dl 的子元素（缩进） |

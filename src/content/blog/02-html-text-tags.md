## 引言

学完标题和段落之后，下一步就是让文字"有变化"——加粗、倾斜、上标、下标、换行等等。

HTML 提供了一组**文本格式化标签**来实现这些效果。但关键问题是：为什么有些标签长得一样（`<b>` 和 `<strong>` 都能加粗），却有两个？

答案还是——**语义化**。

## 加粗：`<strong>` vs `<b>`

```html
<b>这是粗体（纯样式）</b>
<strong>这是强调（语义上重要）</strong>
```

视觉上两者都是加粗，但：

| 标签 | 含义 | 推荐用哪个？ |
|------|------|------------|
| `<b>` | 仅仅是视觉加粗，没有额外含义 | ❌ 不推荐 |
| `<strong>` | 表示内容**很重要**，浏览器加粗显示 | ✅ 推荐 |

> **老师的原话：** "HTML5 之后，`<b>` 和 `<i>` 没有被废弃，但建议用 `<strong>` 和 `<em>`，因为它们有语义。"

## 斜体：`<em>` vs `<i>`

```html
<i>这是斜体（纯样式）</i>
<em>这是强调语气（语义上需重读）</em>
```

同样的逻辑：

| 标签 | 含义 | 推荐用哪个？ |
|------|------|------------|
| `<i>` | 仅仅是视觉斜体（如外来词、术语） | 偶尔用 |
| `<em>` | 表示语气上的强调，屏幕朗读器会重读 | ✅ 推荐 |

## 删除线和下划线

```html
<del>原价 99 元</del> 现价 29 元  <!-- 删除线 -->
<ins>新增内容</ins>                  <!-- 下划线（表示插入） -->
```

`<del>` 表示"已删除"，`<ins>` 表示"新插入"——又是语义化。不要用 `<u>` 来画下划线，因为下划线在网页中通常表示**可点击的链接**，容易误导用户。

## 上标和下标

```html
<p>水的化学式：H<sub>2</sub>O</p>
<p>2 的 3 次方：2<sup>3</sup> = 8</p>
```

显示效果：
- H₂O
- 2³ = 8

这两个标签虽小，但在写技术文章、数学公式或化学式时非常实用。

## 换行和水平线

```html
<p>这是第一行<br />这是第二行</p>
<hr />
<p>分割线上面和下面</p>
```

| 标签 | 作用 | 注意 |
|------|------|------|
| `<br>` | 换行（break） | 是**自闭合标签**，写 `<br>` 或 `<br />` 都可 |
| `<hr>` | 主题分割线（horizontal rule） | 也是自闭合，有语义：表示内容主题的切换 |

> **不要用 `<br>` 来排版！** 如果要段落之间空行，用 CSS 的 `margin`，而不是一堆 `<br><br><br>`。

## 常见的特殊字符（HTML 实体）

在 HTML 中，有些字符是"保留字符"（比如 `<` 和 `>`），直接写会被当成标签解析。这时候需要用**实体名称**或**实体编号**：

```html
<p>&lt; 表示小于号</p>
<p>&gt; 表示大于号</p>
<p>&amp; 表示 and 符号</p>
<p>&nbsp; 表示空格（non-breaking space）</p>
<p>&copy; 表示版权符号 ©</p>
```

常用对照表：

| 显示结果 | 实体名称 | 实体编号 |
|---------|---------|---------|
| `<` | `&lt;` | `&#60;` |
| `>` | `&gt;` | `&#62;` |
| `&` | `&amp;` | `&#38;` |
| 空格 | `&nbsp;` | `&#160;` |
| `©` | `&copy;` | `&#169;` |

老师特意强调了一个**面试常考题**：如果要在网页上显示 `<div>` 这段文本，怎么写？

答案：`&lt;div&gt;`

## 一个综合练习

我把今天学的标签组合起来写了一个小例子：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8" />
  <title>文本标签综合练习</title>
</head>
<body>
  <h1>文本格式化标签练习</h1>
  <p>这是一段<strong>重要内容</strong>，需要强调。</p>
  <p>这是一段<em>需要重读</em>的话。</p>
  <p>水的化学式：H<sub>2</sub>O，2<sup>10</sup> = 1024</p>
  <p><del>已删除的内容</del><ins>新插入的内容</ins></p>
  <p>特殊字符：&lt; &gt; &amp; &copy;</p>
  <hr />
  <p>版权所有 &copy; 2025 我的前端学习笔记</p>
</body>
</html>
```

## 本节小结

| 标签 | 用途 | 语义 |
|------|------|------|
| `<strong>` | 加粗 | ✅ 表示重要 |
| `<em>` | 斜体 | ✅ 表示强调 |
| `<del>` | 删除线 | ✅ 表示已删除 |
| `<ins>` | 下划线 | ✅ 表示新增 |
| `<sub>` | 下标 | ✅ |
| `<sup>` | 上标 | ✅ |
| `<br>` | 换行 | ❌ 无语义，纯样式 |
| `<hr>` | 分割线 | ✅ 表示主题切换 |

## 前言

在 HTML 中，有些字符是**保留字符**——比如 `<` 和 `>` 会被浏览器当成标签的一部分解析。如果你想在页面上**显示**这些字符本身，就需要用**字符实体（Character Entities）** 代替。

字符实体以 `&` 开头、`;` 结尾，格式为：

```
&实体名称;
```

或者

```
&#数字编号;
```

## 一、为什么需要字符实体？

假设你想在页面上显示这段文字：

```
HTML 的 <p> 标签表示段落。
```

如果你直接写：

```html
<p>HTML 的 <p> 标签表示段落。</p>
```

浏览器会把 `<p>` 解析成标签，页面会乱掉。这时候就要用字符实体：

```html
<p>HTML 的 &lt;p&gt; 标签表示段落。</p>
```

## 二、最常用的 5 个字符实体

| 字符 | 名称 | 实体写法 | 数字写法 | 用途 |
|:----:|------|---------|---------|------|
| `<` | 小于号 (less than) | `&lt;` | `&#60;` | 显示 `<` 本身 |
| `>` | 大于号 (greater than) | `&gt;` | `&#62;` | 显示 `>` 本身 |
| `&` | 和号 (ampersand) | `&amp;` | `&#38;` | 显示 `&` 本身 |
| `"` | 双引号 (double quote) | `&quot;` | `&#34;` | 显示 `"` 本身 |
| `'` | 单引号 (single quote) | `&apos;` | `&#39;` | 显示 `'` 本身 |

**最常用的是 `&lt;`、`&gt;` 和 `&amp;`**——在代码块中展示 HTML 标签时几乎一定会用到。

### 示例

```html
<p>&lt;div&gt; 是一个块级容器。</p>
<p>公司名称：AT&amp;T</p>
<p title="他说：&quot;你好&quot;">鼠标悬停看引号效果</p>
```

显示效果：

```
<div> 是一个块级容器。
公司名称：AT&T
```

## 三、常用符号实体

### 空格与排版

| 实体 | 效果 | 说明 |
|------|:----:|------|
| `&nbsp;` | 不换行空格 | 多个连续空格不会被浏览器合并 |
| `&ensp;` | 半角空格 | 宽度是普通空格的两倍 |
| `&emsp;` | 全角空格 | 宽度是 `&ensp;` 的两倍 |
| `&shy;` | 软连字符 | 换行时自动显示连字符 |

`&nbsp;` 最常用——普通空格在 HTML 中多个连续会被合并成一个，用 `&nbsp;` 可以强制显示多个空格。

```html
<p>普通空格：A   B（中间三个空格被合并成一个）</p>
<p>&amp;nbsp;空格：A&nbsp;&nbsp;&nbsp;B（三个空格保留）</p>
```

### 版权与商标

| 实体 | 效果 | 说明 |
|------|:----:|------|
| `&copy;` | © | 版权符号 |
| `&reg;` | ® | 注册商标 |
| `&trade;` | ™ | 商标符号 |

```html
<p>&copy; 2026 版权所有</p>
<p>&reg; 注册商标 &trade; 商标</p>
```

### 数学符号

| 实体 | 效果 | 说明 |
|------|:----:|------|
| `&times;` | × | 乘号 |
| `&divide;` | ÷ | 除号 |
| `&plusmn;` | ± | 正负号 |
| `&sect;` | § | 章节号 |
| `&deg;` | ° | 度（如 30°C） |

```html
<p>3 &times; 4 = 12</p>
<p>10 &divide; 2 = 5</p>
<p>室温：25&deg;C</p>
```

### 货币符号

| 实体 | 效果 | 说明 |
|------|:----:|------|
| `&yen;` | ¥ | 人民币 / 日元 |
| `&euro;` | € | 欧元 |
| `&pound;` | £ | 英镑 |
| `&cent;` | ¢ | 美分 |

## 四、数字编号 vs 实体名称

数字编号可以用在任何 HTML 版本中，实体名称更易读。

```html
<p>&#60; 和 &lt; 都表示小于号 <</p>
<p>&#169; 和 &copy; 都表示版权符号 ©</p>
```

两者效果完全一样，推荐使用**实体名称**，更容易记忆。

## 五、什么时候必须用？

| 场景 | 必须用？ | 举例 |
|------|---------|------|
| 显示 `<` 和 `>` | ✅ 必须 | `&lt;div&gt;` |
| 显示 `&` | ✅ 必须 | `AT&amp;T` |
| 显示连续空格 | ✅ 推荐 | `&nbsp;&nbsp;&nbsp;` |
| 显示版权/商标 | ✅ 推荐 | `&copy; 2026` |
| 显示引号在属性中 | ✅ 推荐 | `&quot;` |
| 显示普通文字 | ❌ 不用 | 直接写即可 |

## 小结

字符实体是 HTML 中用来**安全显示特殊字符**的方法，核心要点：

1. `&lt;`、`&gt;`、`&amp;` 是最常用的三个——**写代码展示时必用**
2. `&nbsp;` 用来保留连续空格
3. `&copy;`、`&reg;`、`&trade;` 是版权相关的符号
4. `&times;`、`&divide;`、`&deg;` 是常用数学符号
5. 掌握这十几个常用的字符实体，日常开发就够了

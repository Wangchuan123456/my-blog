## 前言

学完标题和段落之后，老师开始讲"块级元素"和"行内元素"的概念。听完之后发现，这是理解 HTML 布局的**关键一步**——搞懂了这两个，就知道为什么某些标签能套在一起、某些不能。

## 什么是块级元素？

**块级元素（Block-level elements）**的特点：

1. **独占一行**，前后有换行
2. 默认宽度占满父容器
3. 可以设置宽高（`width` / `height`）

常见的块级元素：

| 标签 | 含义 |
|------|------|
| `<h1>` ~ `<h6>` | 标题 |
| `<p>` | 段落 |
| `<div>` | 无语义容器（最常用） |
| `<ul>` / `<ol>` | 列表 |
| `<hr>` | 水平分割线 |

```html
<h1>我是块级元素</h1>
<p>我也是块级元素</p>
```

这两个标签在浏览器里会**各自独占一整行**。

## 什么是行内元素？

**行内元素（Inline elements）**的特点：

1. **不会换行**，多个行内元素排在一行
2. 宽度由内容决定
3. **不能设置宽高**（上下 padding / margin 也容易出问题）

常见的行内元素：

| 标签 | 含义 |
|------|------|
| `<span>` | 无语义容器 |
| `<strong>` | 加粗强调 |
| `<em>` | 斜体强调 |
| `<del>` | 删除线 |
| `<mark>` | 高亮 |
| `<a>` | 链接 |

```html
<span>我是行内元素</span>
<strong>我也是行内的</strong>
<em>我也一样</em>
```

这三个会**排在同一行**显示。

## 嵌套规则（重点）

这是老师强调的核心规则，也是写 HTML 最容易犯错的地方。

### 规则 1：块级元素可以包含行内元素 ✅

```html
<p>
  <strong>我是加粗的</strong>
  <em>我是斜体</em>
</p>
```

`<p>` 是块级，`<strong>` 和 `<em>` 是行内——**可以这样嵌套**。

### 规则 2：行内元素可以嵌套行内元素 ✅

```html
<del>
  <em>我是斜体且带删除线</em>
  <strong>
    <mark>我是加粗又高亮</mark>
  </strong>
</del>
```

`<del>` 是行内，里面再套 `<em>`、`<strong>`、`<mark>`，全部都是行内——**可以这样嵌套**。

### 规则 3：行内元素不能包含块级元素 ❌

```html
<!-- 错误写法 -->
<span>
  <p>段落不能放在 span 里面</p>
</span>
```

这是**无效的 HTML**。行内元素的"盒子"装不下块级元素，浏览器虽然会尽量渲染，但可能会自动把 `<p>` 踢到外面去。

### 规则 4（特殊情况）：`<p>` 标签不能包含其他块级元素 ❌

```html
<!-- 错误写法 -->
<p>
  <h2>标题不能放在段落里</h2>
</p>
```

`<p>` 是一个**特殊的块级元素**——它的内容模型（content model）只允许** phrasing content（措辞内容，即行内元素）**。如果你在里面放块级元素，浏览器会自动闭合 `<p>`，导致结构错乱。

## 我练习的代码

这是我自己写的例子，验证了上面几条规则：

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>块级与行内练习</title>
</head>

<body>
  <!-- 块级元素 -->
  <h1>我是块级元素</h1>
  <p>我也是块级元素</p>

  <!-- 行内元素 -->
  <span>我是行内元素</span>

  <!-- 块级元素能够包含行内元素 -->
  <p>
    <strong>我是加粗的</strong>
  </p>

  <!-- 行内元素可以嵌套其他行内元素 -->
  <del>
    <em>我是斜体的</em>
    <strong>
      <mark>我是高亮的</mark>
    </strong>
  </del>

  <!-- 特殊情况，p标签不能包含其他块级元素 -->

</body>

</html>
```

## 记忆小技巧

老师教了一个很好的记忆方法：

> **块级元素**就像一个大箱子，可以放小盒子（行内元素），也可以放大箱子（其他块级元素）。
>
> **行内元素**就像小盒子，不能放大箱子，但可以放更小的盒子（其他行内元素）。
>
> **`<p>` 比较特殊**——它虽然是大箱子，但里面只能放小盒子，不能放其他大箱子。

## 小节总结

| 规则 | 例子 | 是否允许 |
|------|------|---------|
| 块级 ⊃ 行内 | `<div><span>...</span></div>` | ✅ |
| 块级 ⊃ 块级 | `<div><p>...</p></div>` | ✅（`<p>` 除外） |
| 行内 ⊃ 行内 | `<strong><em>...</em></strong>` | ✅ |
| 行内 ⊃ 块级 | `<span><p>...</p></span>` | ❌ |
| `<p>` ⊃ 块级 | `<p><div>...</div></p>` | ❌ |

搞懂这些，写 HTML 时就不会出现"浏览器渲染出来跟想的不一样"的困惑了 🚀

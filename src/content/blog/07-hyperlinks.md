## 前言

超链接是网页最核心的功能——没有链接，互联网就不是"网"。HTML 中用 `<a>` 标签来创建超链接，它的核心属性是 `href`（链接目标地址）。

## 超链接的 5 种类型

### 1. 内部链接

指向**同一网站内**的其他页面：

```html
<a href="sub/other-page.html">跳转到内部页面</a>
<a href="/pages/about.html">根相对路径引用</a>
```

内部链接用**相对路径**或**根相对路径**来写，和之前学的图片路径规则完全一样。

### 2. 外部链接

指向**其他网站**的链接，必须写完整的 URL（以 `https://` 开头）：

```html
<a href="https://www.baidu.com" target="_blank">去百度搜索</a>
<a href="https://github.com" target="_blank">去 GitHub</a>
```

**关于 `target="_blank"`：** 加上这个属性后，点击链接会在**新标签页/新窗口**中打开，用户当前浏览的页面不会被替换。外部链接建议都用 `target="_blank"`，避免用户离开你的网站。

### 3. 空链接

用于**占位**，链接还没有确定目标时先用空链接代替（通常后续用 JavaScript 绑定功能）：

```html
<!-- 写法一：点击回到页面顶部（href="#" 会跳到顶部） -->
<a href="#">空链接（占位用）</a>

<!-- 写法二：点击不产生任何效果 -->
<a href="javascript:;">空链接（不跳转）</a>
```

**两种写法的区别：**
- `href="#"`：点击后页面会滚动到顶部，`#` 默认指向页面最顶部
- `href="javascript:;"`：点击后什么都不会发生

开发中根据需求选择，如果不需要"回到顶部"的效果，就用 `javascript:;`。

### 4. 下载链接

点击后直接下载文件，而不是在浏览器中打开：

```html
<a href="../1.jpg" download>下载示例图片</a>
<a href="../horse.mp3" download>下载示例音频</a>
```

关键就是加上 `download` 属性——它是一个**布尔属性**（键值相同可省略），告诉浏览器"不要打开这个文件，直接下载它"。

### 5. 邮件链接

点击后打开系统默认的邮件客户端，收件人自动填好：

```html
<!-- 基本用法 -->
<a href="mailto:example@example.com">发送邮件</a>

<!-- 带主题和正文 -->
<a href="mailto:example@example.com?subject=前端学习咨询&body=你好，我想咨询前端学习的问题。">
  发送邮件（预填主题和正文）
</a>
```

**`mailto:` 的参数：**
- `?subject=主题` — 预填邮件主题
- `&body=正文` — 预填邮件正文（用 `&` 连接多个参数）
- `&cc=抄送地址` — 抄送
- `&bcc=密送地址` — 密送

## 锚点链接：同一页面内跳转

锚点链接让用户点击后**跳转到当前页面的某个位置**，非常适合长页面的"目录导航"和"回到顶部"。

### 用法分两步

**第 1 步：给目标元素设置 id**

```html
<h2 id="links-section">超链接示例</h2>
```

`id` 是元素的**唯一标识符**，同一个页面中不能有重复的 id。

**第 2 步：用 `#id` 指向目标**

```html
<!-- 点击后页面滚动到 id="links-section" 的位置 -->
<a href="#links-section">跳转到超链接示例</a>
```

`href` 里的 `#` 加上目标 `id` 的值，就是锚点链接的完整写法。

### 常见用法

**回到顶部：**

```html
<a href="#">⬆ 回到顶部</a>
```

`#` 不带任何 id 时，默认跳转到页面顶部——和空链接的写**法一样**，但用途不同：空链接是占位，这个是真的要跳转到顶部。

**跨页面锚点：**

```html
<!-- 先跳转到 other.html，再定位到 id="section2" 的位置 -->
<a href="other.html#section2">跳转到其他页面的指定位置</a>
```

## 属性总结

| 属性 | 作用 | 示例 |
|------|------|------|
| `href` | 链接目标地址 | `href="https://www.baidu.com"` |
| `target="_blank"` | 新窗口打开 | 外部链接常用 |
| `download` | 触发下载 | 下载文件时添加 |
| `id` | 锚点目标标识 | `id="section1"` |

锚点链接和超链接用的是同一个 `<a>` 标签，区别只在于 `href` 写的是 `#id` 还是 URL——理解了这个，所有链接的用法就都掌握了。

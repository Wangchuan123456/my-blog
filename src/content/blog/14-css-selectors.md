## 前言

CSS 选择器（Selector）的作用是**选中页面上的元素**，然后给它们设置样式。

> 可以把选择器理解为"寻人启事"——描述你要找的人有什么特征，浏览器就去页面上找到匹配的元素。

最基础的四种选择器：

| 选择器 | 写法 | 选中什么 |
|--------|------|---------|
| **类型选择器** | `p`、`div`、`h1` | 所有同类型标签 |
| **类选择器** | `.class-name` | 所有带此 class 的元素 |
| **ID 选择器** | `#id-name` | 唯一带此 id 的元素 |
| **通配符选择器** | `*` | 所有元素 |

## 一、类型选择器（Type Selector）

也叫**标签选择器**或**元素选择器**，直接写 HTML 标签名，选中**所有该类型的元素**。

```css
p {
  color: green;
}

strong {
  color: red;
}

h1 {
  font-size: 24px;
}
```

```html
<p>我喜欢绿色</p>
<p>我也喜欢绿色</p>
<strong>我喜欢红色</strong>
<strong>我也喜欢红色</strong>
```

效果：所有 `<p>` 都变绿色，所有 `<strong>` 都变红色。

### 特点

| 项目 | 说明 |
|------|------|
| 写法 | 直接写标签名，如 `p`、`div`、`h1` |
| 选中范围 | **所有**同名标签 |
| 优先级 | 低（权重 0,0,1） |
| 适用场景 | 设置全局默认样式 |

### 什么时候用？

- 想给**所有**某个标签设置统一样式时
- 比如：所有 `<p>` 统一行高、所有 `<a>` 去掉下划线

## 二、类选择器（Class Selector）

最常用、最灵活的选择器。通过 HTML 的 `class` 属性给元素命名，用 `.` 选中。

```css
.highlight {
  background: yellow;
  font-weight: bold;
}

.card {
  border: 1px solid #ddd;
  padding: 1rem;
  border-radius: 8px;
}
```

```html
<p class="highlight">这段文字会被高亮</p>
<p>这段不会变</p>
<div class="card">这是一个卡片</div>
<div class="card">另一个卡片</div>
```

### 特点

| 项目 | 说明 |
|------|------|
| 写法 | `.类名`，如 `.highlight` |
| 选中范围 | **所有**带此 class 的元素 |
| 优先级 | 中等（权重 0,1,0） |
| 适用场景 | **日常开发最常用** |

### 关键特性

1. **一个元素可以有多个 class**，用空格隔开：
   ```html
   <div class="card highlight">既有卡片样式，又有高亮背景</div>
   ```

2. **多个元素可以共用同一个 class**：
   ```html
   <p class="warning">警告信息</p>
   <div class="warning">另一个警告</div>
   ```

3. **class 名不能以数字开头**，推荐用 `-` 分隔：`my-class`

### 什么时候用？

- **日常开发中绝大部分场景**
- 需要给某些元素设置相同样式时
- 需要把一个样式复用到多个元素时

## 三、ID 选择器（ID Selector）

通过 HTML 的 `id` 属性给元素一个**唯一标识**，用 `#` 选中。

```css
#header {
  background: #333;
  color: white;
  padding: 1rem;
}

#main-title {
  font-size: 2em;
  color: #6c63ff;
}
```

```html
<div id="header">网站标题</div>
<h1 id="main-title">欢迎来到我的博客</h1>
```

### 特点

| 项目 | 说明 |
|------|------|
| 写法 | `#id名`，如 `#header` |
| 选中范围 | **唯一**（一个页面中 id 不能重复） |
| 优先级 | 最高（权重 1,0,0） |
| 适用场景 | 页面中唯一的元素（如页头、导航） |

### 重要规则

> **一个页面中，同一个 id 只能出现一次。** 这是 HTML 规范，违反了会导致 JavaScript 行为异常。

```html
<!-- ✅ 正确：每个 id 唯一 -->
<div id="header">页头</div>
<div id="footer">页脚</div>

<!-- ❌ 错误：id 重复 -->
<div id="nav">导航</div>
<div id="nav">另一个导航</div>
```

### class vs id 怎么选？

| 场景 | 用 class | 用 id |
|------|:--------:|:-----:|
| 多个元素共用一个样式 | ✅ | ❌ |
| 唯一元素（如页头、页脚） | ✅ 也可以 | ✅ 适合 |
| 需要 JS 操作 | ✅ | ✅ |
| 优先级更高 | ❌ | ✅ |

> **推荐优先用 class**，只有明确是"页面唯一"且"需要高优先级"时才用 id。

## 四、通配符选择器（Universal Selector）

用 `*` 表示，选中**页面上的所有元素**。

```css
/* 所有元素去掉默认内外边距 */
* {
  margin: 0;
  padding: 0;
}

/* 所有元素的盒子模型统一 */
* {
  box-sizing: border-box;
}
```

### 特点

| 项目 | 说明 |
|------|------|
| 写法 | `*` |
| 选中范围 | **所有元素** |
| 优先级 | 最低（权重 0,0,0） |
| 适用场景 | 全局重置样式 |

### 什么时候用？

- **CSS 重置**——清除浏览器默认样式
- 设置全局统一的 `box-sizing`
- **不要过度使用**——因为会匹配每个元素，大量样式规则会影响性能

## 五、四种选择器权重对比

CSS 选择器有**权重**（Specificity）概念——权重越高，样式优先级越大。

| 选择器 | 示例 | 权重 |
|--------|------|:----:|
| 通配符 | `*` | 0,0,0 |
| 类型选择器 | `p`、`div` | 0,0,1 |
| 类选择器 | `.class` | 0,1,0 |
| ID 选择器 | `#id` | 1,0,0 |
| 行内样式 | `style=""` | 1,0,0,0 |

当多个选择器选中同一个元素时，**权重高的生效**。如果权重相同，**后写的覆盖先写的**。

```css
p { color: green; }         /* 权重 0,0,1 */
.text { color: blue; }      /* 权重 0,1,0 —— 比上面高，生效 */
```

```html
<p class="text">我是什么颜色？</p>  <!-- 蓝色，因为类选择器权重更高 -->
```

## 六、选择器使用建议

| 场景 | 推荐选择器 |
|------|-----------|
| 统一全部元素的边距 | 通配符 `*` |
| 所有标题的字体 | 类型选择器 `h1`、`h2` |
| 多个元素同一种样式 | 类选择器 `.card` |
| 页面中唯一的区块 | ID 选择器 `#header` |
| 覆盖其他样式 | ID 选择器或行内样式 |

## 小结

1. **类型选择器**（`p`）— 选所有同名标签，用于设置全局默认样式
2. **类选择器**（`.class`）— 选所有带此 class 的元素，**日常最常用**
3. **ID 选择器**（`#id`）— 选唯一元素，优先级最高
4. **通配符选择器**（`*`）— 选所有元素，用于全局重置

> **日常开发中，80% 的情况用类选择器就足够了。** 简洁、灵活、复用性强。

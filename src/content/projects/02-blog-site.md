## 项目简介

你现在正在看的这个博客，就是我学习前端的**第一个正式项目**。

选择 Astro 的原因：
- **支持 Markdown 写文章**：专注内容，不用管页面
- **零 JS 默认输出**：页面加载快
- **Content Collections**：每篇文章自动校验格式

## 学到了什么

1. **项目搭建**：从 `npm create astro` 到完整的项目结构
2. **内容管理**：用 Content Collections 统一管理博客和项目
3. **CSS 变量**：实现暗色模式切换
4. **部署流程**：Git 提交 → GitHub → Vercel 自动部署

## 博客会记录什么

- 📝 **学习笔记**：每节课的总结和踩坑记录
- 🚀 **项目展示**：学习过程中的所有作品
- 💡 **心得体会**：学习方法、面试准备、求职记录

这是一个陪我一起成长的博客。内容会不断更新 💪
